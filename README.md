# SuperMario
A Super Mario Bros DQN player based on https://github.com/marblexu/PythonSuperMario

# Requirement
* Python 3.9.12
* Python-Pygame 2.5.2
* Python-Tensorflow 2.15.0
* Python-Numpy 1.26.2

# How To Start Game
$ python main.py
